## CloudTempo Alpha

CloudTempo is currently in alpha. This means that it is not available in the Chrome Web Store or any other official extensions store yet. You can still install it, but you will need to do so manually.

### Instructions

1. Download the [latest version of CloudTempo extension](/download)
2. Unzip the downloaded `extension.zip` file
3. Open Chrome and go to `chrome://extensions`
4. Enable developer mode
   ![/install-developer-mode.png](/install-developer-mode.png)

5. Click on `Load unpacked` and select the unzipped folder
   ![/install-load-unpacked.png](/install-load-unpacked.png)

6. Go to [AWS Console](https://console.aws.amazon.com/console/home) and press `CMD + K` (on Mac) or `CTRL + K` (on Windows/Linux) to open the CloudTempo.
